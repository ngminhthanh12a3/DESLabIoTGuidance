DESLab IoT Training
===================

Đây là nội dung training hệ thống IoT của phòng thí nghiệm DESLab. Trong tài liệu training, tài liệu sẽ hướng dẫn xây dựng hệ thống IoT đơn giản. Hệ thống IoT sẽ giám sát nhiệt độ phòng thông qua cảm biến LM35_, module `Arduino ESP8266 Wifi Shield <AEWS_>`_, và `Arduino Cloud <ACloud_>`_.
